# 有限オートマトンと正規表現

## 基本モデル
- `automata.py`
    - 決定性有限オートマトン: `DFA`
    - 非決定性有限オートマトン: `NFA`
    - 非決定性有限オートマトンから決定性有限オートマトンへ変換: `NFA2DFA`
- `regularGrammar.py`
    - 正規文法: `RG`

- 非決定性有限オートマトン

- 非決定性FAから決定性FAへの変換
